# Invoke-YFFileVersionCleanup

## SYNOPSIS
Removes old file versions from the YF site.

## SYNTAX

```
Invoke-YFFileVersionCleanup [-NoTelemetry] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Removes old file versions from the YF site.

## EXAMPLES

### Example 1

```powershell
PS C:\> Invoke-YFFileVersionCleanup
```
